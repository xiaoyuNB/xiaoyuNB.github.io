    JDesktopPane如何添加背景图片

    Java Java JDesktopPane 添加背景

    字数:579 阅读:

    ####最近小编一直在做项目，准备蓝桥杯，准备期末考试，所有一直没有时间更新博文，今日正值周末，小编于图书馆做项目期间，出现了小问题，并解决了，特地来简书更新一波….

    在javaSwing窗口中，我们常使用JdesktopPane面板，它是一个可以装载JInternalFrame的容器。
    为了整体界面美观，小编在设计JdesktopPane面板时，发现JdesktopPane可以设置背景颜色，但是没有直接为JdesktopPane设置背景图片的方法，也不可以给JdesktopPane添加icon。

    左思右想，小编想出了一个好主意，就是给JdesktopPane面板添加一个JLable标签，将这个JLable标签大小与JdesktopPane面板大小设置一致，再给这个JLable标签添加icon，就能成功的给JdesktopPane面板添加背景图片啦！（效果见下图）

    高兴之余，小编随即点开了上方的菜单项，发现JInternalFrame子面板不显示了！顿时小编傻了眼，仔细一看在发现，原来设置背景图片的JLable标签，会覆盖住已经弹出的JInternalFrame子面板。也就是说，JInternalFrame会被JLable标签覆盖住。小编又傻了。。。

    在游览了各大论坛和各个大神的技术贴后，我大概知道要修改叠放层次，或者说修改面板与组件之间的显示优先级。但是论坛大神们总是点睛之笔，或者随手丢下三两行代码，让人依然摸不着头脑。。

    所谓物极必反，无奈之下，小编灵机一动，想出了一个办法：每当我点开一个JInternalFrame子容器时，我添加一个点击事件，只要弹出JInternalFrame面板，就立即设置JLable背景标签隐藏（不可见）。小编亲测有效，疑难杂症终于搞定了。。。（效果见下图）
