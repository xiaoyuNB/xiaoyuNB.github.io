    • 2.1 Java的基本语法

    • 标识符不能以数字开头，不能是关键字
    • const和goto是保留关键字，暂时没有意义，但不能用作标识符
    • true,false和null 不属于关键字，是单独的标识类型，不能直接使用
    • 二进制以0b或0B开头，八进制以0开头，十六进制以0x或0X开头
    • \u0000 是一个空白字符 属于Unicode字符集

    2.2 Java中的变量

    • 数据类型分为基本数据类型和引用数据类型，引用类型分为（类、接口、数组、枚举、注解）

    • 1.4E-45表示1.4*10-45

    • float类型的变量赋值，后面必须加上字母F（或者小写f） 如：float a = 134.5f

    • char赋值整数，会自动转换成字符 如：char c = ‘a’; char ch =97; 打印出来的结果都是a

    • 变量之间类型转换的基本原则：大吃小、浮点吃整数、double通吃

    • 数值溢出：大转小 精度丢失： int转byte int a=298转为byte a，byte a =298-256=42

    • byte型变量在参与运算期间，自动提升为int型

    2.3 Java中的运算符

    • ++a 加号在前参与运算；a++ 加号在后不参与运算，无论在前在后，a的值依然自增
    • 整数之间除法运算，会忽略小数 如2510/1000=2；除法直接有小数参与，会计算小数 如2.5/10=0.25
    • 取模（%）运算时，运算结果的正负取决于被模数（%左边的数） 如：(-5) %3= -2；5 % (-3)= 2
    • 用+=、-+、*=、/=、%= 运算符进行赋值时，强制类型转换会自动完成
    • 比较运算符== 赋值运算符 =
    • &和&&：一假且为假，但&两边表达式都会进行运算，而&&当左边表达式为false时，右边表达式不进行运算
    • |和||：一真或为真 ，同上，当||左边表达式为true时，右边表达式不会进行运算
    • ^ 异或：两边都为真或两边都为假，则为false；一边真一边假即为true

    2.4 选择结构语句

    • s三元运算： 判断条件 ？ 表达式1 ：表达式2 如 ：int max = x> y? x: y
    • 新版JDK特性：switch语句可以接收String类型的值

    2.5 循环语句

    • for循环：for（1；2；3）{ 4 }

      第一步，执行1

      第二步，执行2，如果判断结果为true,执行第三步，如果判断结果为false，执行第五步

      第三步，执行4

      第四步，执行3，然后重复执行第二步

      第五步，退出循环

    • 用itcast做标记，break itcast跳出外层循环；continue语句后面也可以通过使用标记的方式结束本次外层循环

    • 解决输入问题：Scanner sc =new Scanner (System.in);

      ​ int i = sc.nextInt( );

    • Random类中的nextInt( int n)方法，生成一个0-9的随机数：

      int randomNumber = new Random( ).nextInt(10);

    • break：终止循环，跳出循环，执行后续代码

      continue：跳出本次循环，后续的循环继续执行，循环执行完毕，后续的代码，继续执行

      return：直接终止当前运行的方法

    2.6 方法

    • 方法就是函数

    • void没有返回值，其余必须有返回值

    • 重载满足两个条件：一是方法名相同，而是参数的个数或者参数的类型不同。注： 重载与返回值无关

    • 一维数组定义： int [ ] x=new int [100];

    • 数组.length 获得数组的长度，即元素的个数

    • 索引（数组每个元素角标）的范围是：0到 length-1

    • 数组变量必须指向一个有效的数组对象，不能为null，否则会报空指针异常NullPointerException

    • 多维数组： int [ ] [ ] arr = new int [3] [4]; 3*4的二维数组
